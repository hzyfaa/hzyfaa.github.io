Artifact Overview

The artifact that I chose is a full-stack web application that I developed as part of my coursework for CS 465: Full-Stack Development in my previous term. The application is a travel web application developed using the MEAN (MongoDB, Express, Angular, and Node.js) stack framework. The features include a customer-facing website where users could view a list of different travel packages and an admin portal for managing these travel packages. Additionally, the admin page includes a login portal to authorize users before they can edit or add travel packages.
